# Investment-Advisor

## Objectives of the Project:

Objective 1: To update the data in Google Sheets using Python.

Objective 2: To suggest a KPI that can be used to pick stocks from different sectors.



## Tools Used:

Google API

Python Libraries - Gspread, Pandas

Excel
